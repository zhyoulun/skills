# WeChat Article Selectors Reference

- **Title**: `h1.rich_media_title`
- **Main Body**: `div#js_content`
- **Body Wrapper**: `div.rich_media_content`
- **Author**: `span.rich_media_meta_text` (sometimes) / `#js_name` (Account name)
- **Images**: `img` tags typically use `data-src` for the actual URL.
- **Videos**: Usually embedded as `iframe`.
